# Biblioteca em Java com Interface Gráfica

Projeto acadêmico desenvolvido em Java com interface gráfica (GUI).

## Objetivo
Praticar conceitos de programação orientada a objetos, organização de código
em classes e interação com o usuário por meio de interface gráfica.

## Tecnologias
- Java
- Interface Gráfica (GUI)

Projeto desenvolvido durante a graduação em Ciência da Computação.
